# 🛡️ Bybit Sniper Bot v3.0 (Volume Hunter Edition)

Професійний торговий інструмент для автоматизації торгівлі на **Bybit (Demo/Real)**, що працює на основі аналізу об'ємних аномалій та пробоїв локальних діапазонів.

## 🛠 Основні функції
* **Volume Spike Detection:** Вхід у позицію лише при сплеску об'єму (x1.8+ від середнього).
* **Hard Risk Management:** Фіксована маржа (5 USDT), плече 20x.
* **Pure Math:** Співвідношення Risk/Reward — **1:2** (SL 1.5% / TP 3.0%).
* **Data Science Ready:** Розширене логування тех-даних (`Relative Volume`, `Hour`) у SQLite для подальшого аналізу.
* **Admin Tools:** Кнопки очищення статистики та експорту бази даних прямо в Telegram.

## 📊 Стратегія (V.I.R.B.)
Бот використовує логіку **VSA (Volume Spread Analysis)**, ігноруючи запізнілі індикатори:
1.  **Таймфрейм:** M15 (мінімізація ринкового шуму).
2.  **Аналіз:** Розрахунок середнього об'єму за останні 20 свічок.
3.  **Сигнал:** Закриття свічки вище/нижче екстремумів останніх 5 свічок, підтверджене аномальним об'ємом.
