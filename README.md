# 🚀 Bybit Sniper Bot v2.0 (Fixed Risk Edition)

Професійний торговий бот для Bybit (Demo/Real) з фокусом на ризик-менеджмент та детальну аналітику.

## 🛠 Основні функції
- **Fixed Risk:** Вхід строго на 5 USDT маржі з плечем 20x (автоматичний розрахунок лота).
- **Auto-Breakeven:** Перенос стоп-лоссу в ціну входу при досягненні +1.2% профіту.
- **Analytics Engine:** Збереження історії угод у SQLite для аналізу профіту по годинах та монетах.
- **Telegram Control:** Повне керування кнопками (Баланс, Поточні угоди, Денний звіт, Panic Sell).

## 📊 Стратегія
- **Індикатори:** EMA 13, 34, 89 (Тренд-фолловінг).
- **Таймфрейм:** 5 хвилин.
- **Трейлінг-стоп:** 1% від поточної ціни.

## 🚀 Швидкий запуск
1. Встановіть залежності: `pip install -r requirements.txt`
2. Налаштуйте Environment Variables: `API_KEY`, `API_SECRET`, `TELEGRAM_TOKEN`, `TELEGRAM_CHAT_ID`.
3. Запустіть: `python main.py`

## 📈 Кнопки в Telegram
- `💰 Баланс` — реальний стан рахунку на Bybit.
- `📊 Поточні угоди` — ROI та PnL відкритих позицій.
- `📈 Статистика монет` — рейтинг найбільш прибуткових активів.
- `📅 Денний звіт` — чистий результат за останні 24 години.
- `⚠️ PANIC SELL` — миттєве закриття всіх позицій по ринку.
