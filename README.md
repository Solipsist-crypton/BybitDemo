# 🎯 Crypto Sniper Bot (Bybit V5)

Мультимонетний бот для торгівлі на ф'ючерсах (Demo/Main) за стратегією **EMA 13/34/89**.

## 🚀 Функціонал
- **Монети**: BTC, ETH, SOL, XRP, ADA.
- **Аналіз**: Таймфрейм 5 хвилин (M5).
- **Сповіщення**: Повні звіти про угоди в Telegram.
- **Безпека**: Використання змінних оточення (Variables).

## 🛠 Встановлення
1. Клонувати репозиторій.
2. Встановити залежності: `pip install -r requirements.txt`.
3. Налаштувати змінні (`API_KEY`, `API_SECRET`, `TELEGRAM_TOKEN`, `TELEGRAM_CHAT_ID`) в панелі сервера.
4. Запустити: `python main.py`.
